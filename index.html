CONECTAONGs

Este agregador de ONGs foi desenvolvido com o intuito de consolidar e aprimorar minhas habilidades em HTML e CSS. Utiliza conceitos como Flex e Grid para criar layouts flexíveis e adaptáveis a diferentes dispositivos e eventos interativos utilizando javaScript.
Através deste projeto, busquei boas práticas de código como o HTML semântico, a estilização pra vários dispositivos atraves de media queries e a construção de uma interface intuitiva, acessível e agradável ao usuário. Além de

Este projeto foi realizado como adaptação de um projeto sugerido no módulo de CSS (ministrado pelo professor web full-stack Felipe Cabral ) na trilha front-end, que é parte do Santander Tech + realizado como uma parceria entre o banco Santander e a empresa Ada Tech.

As ONGs desempenham um papel fundamental em nossa sociedade, atuando em diversas áreas como assistência social, educação, saúde, meio ambiente, direitos humanos e proteção animal. Estas organizações sem fins lucrativos trabalham incansavelmente para preencher lacunas importantes onde a atuação governamental pode ser limitada, proporcionando serviços essenciais e apoio às comunidades mais vulneráveis. A diversidade de ONGs permite que diferentes causas sejam atendidas, cada uma com sua especialidade e foco específico, contribuindo para um impacto social mais amplo e efetivo em nossa sociedade.
